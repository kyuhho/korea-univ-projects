# Korea University Projects

A collection of projects from Korea University coursework.

## XAI507: Computer Vision

- **[Project 1](./XAI507_Project1)**
  - **Topic**: Image Processing & Restoration
  - **Description**: Implementation of image degradation (blur, noise, low contrast) and restoration techniques including denoising and deblurring.

- **[Project 2](./XAI507_Project2)**
  - **Topic**: Vision Transformers (ViT)
  - **Description**: Analysis and design of positional encoding methods (Sinusoidal, RoPE, Learnable) within Vision Transformers.

## XAI625: Reinforcement Learning Application and Practice

- **[Final Project](./XAI625_Final_Project)**
  - **Topic**: Monte Carlo Tree Search (MCTS)
  - **Description**: A Tic-Tac-Toe agent instantiated using the MCTS algorithm, featuring AI gameplay and search tree visualization.
